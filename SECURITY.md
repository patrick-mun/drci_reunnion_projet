# Security Policy

## Supported Versions

| Version | Supported |
| ------- | --------- |
| main    | ✅ |

## Reporting a Vulnerability

If you discover a security vulnerability, please do not open a public issue.
Report it privately to: patrick.munier@gmail.com

I will acknowledge receipt as soon as possible and assess the report before any public disclosure.
